HTML & CSS Quick Reference

Your go-to guide for essential tags and styles

1. Basic HTML Tags

Here are some essential HTML tags:

        <h1></h1>
        <p></p>
        <a href="#"></a>
        <img src="image.jpg" alt="Description">

Output:

This is an H1 heading

This is a paragraph of text.

2. Lists and Formatting

Useful for organizing content:

        <ul><li></li></ul>
        <ol><li></li></ol>
        <strong></strong>
        <em></em>

Output:

  • Unordered List Item 1
  • Unordered List Item 2
  1. Ordered List Item 1
  2. Ordered List Item 2

This text is bold. This text is italicized.

3. Example: <div> Tag

The <div> tag is a container for other HTML elements. It's often used for grouping and styling purposes.

        <div class="example">Content goes here</div>

Output:

Content goes here

4. <span> Tag

The <span> tag is an inline container used to apply styles or scripting to a specific portion of text.

        <p>This is a <span class="highlight">highlighted</span> word.</p>

Output:

This is a highlighted word.

5. <input> Tag

The <input> tag is used to create interactive form controls.

        <input type="text" placeholder="Enter your name">

Output:

6. <button> Tag

The <button> tag defines a clickable button, which can be used in forms or to trigger JavaScript functions.

        <button onclick="alert('Hello!')">Click me</button>

Output:

7. <hr> Tag

The <hr> tag represents a horizontal rule, used to separate content.

        <p>Some text above the line.</p><hr><p>Some text below the line.</p>

Output:

Some text above the line.

Some text below the line.

8. <br> Tag

The <br> tag inserts a line break within text.

        <p>This is the first line.<br>This is the second line.</p>

Output:

This is the first line.
This is the second line.

9. <iframe> Tag

The <iframe> tag is used to embed external content, such as a web page or video.

        <iframe src="https://www.example.com" width="400" height="300"></iframe>

Output:

          <select>

            <option value="option2">Option 2</option>

Additional Styles

This section demonstrates additional CSS properties:

box-shadow:

Adds a shadow effect to an element.

box-shadow: 0 0 15px rgba(255, 255, 255, 0.3);

font-size:

Sets the font size of an element's text.

font-size: 1.2em;

margin-top:

Specifies the margin on the top of an element.

margin-top: 20px;

padding:

Defines the padding inside an element.

padding: 30px;

background-color:

Sets the background color of an element.

background-color: #555;

border-radius:

Creates rounded corners for an element.

border-radius: 15px;

line-height:

Sets the height of a line of text.

line-height: 1.5;

text-decoration:

Controls the decoration of text (underline in this case).

text-decoration: underline;
